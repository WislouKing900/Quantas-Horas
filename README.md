# Quantas-Horas
Estou apresentando a Calculadora de Horas, uma ferramenta que desenvolvi para otimizar o controle de tempo.

Qual o objetivo?

O principal objetivo é eliminar a necessidade de cálculos manuais, que são propensos a erros e consomem tempo valioso. A ferramenta permite que você insira o horário de início e término de uma atividade, e ela automaticamente calcula a duração total em horas.
